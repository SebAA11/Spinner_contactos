# 📒 Agenda de Contactos Web

Aplicación web desarrollada con **HTML, CSS y JavaScript puro** que permite gestionar una lista de contactos de manera sencilla, almacenando la información en el **LocalStorage del navegador**.

---

## 🚀 Características

* Registro de contactos con los siguientes campos:

  * Nombre
  * Apellido
  * Ciudad
  * Teléfono
  * Dirección
  * Género

* Visualización dinámica de contactos

* Eliminación individual de registros

* Persistencia de datos usando LocalStorage

* Interfaz moderna y responsiva

* Validación básica de campos

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* LocalStorage (Web API)

## 💾 Funcionamiento del almacenamiento

La aplicación utiliza **LocalStorage**, por lo tanto:

* Los datos se guardan en el navegador
* No se requiere base de datos
* La información persiste incluso al recargar la página
* Los datos son específicos por navegador y dominio
